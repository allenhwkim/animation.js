Basic Example

Custom setting to control timing and duration

For a website, it's recommened to have the same duration and timing function for all animations. 
import {Animation} from 'animations.js';
Animation.DURATION = 1000;
Animation.TIMING = 'ease-in-out';

For Fully Customed Animation

To fully control your animation, run `animate` function 
import {animate} from 'animations.js';

const el = document.querySelector('.target');

// run draw function only
animate(pct => el.style.opacity = pct);

// or durarion, timing as string, and draw function
animate(1000, 'ease-in-out', pct => el.style.opacity = pct);

// or or durarion, timing function, and draw function
animate(
  1000,                          // DURATION
  t => Math.pow(t, 1.675),       // timing function
  pct => el.style.opacity = pct  // draw function  
)
.then(_ => el.removeAttribute('style'));
